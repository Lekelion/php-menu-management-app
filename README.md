# PHP Menu Management App

A PHP/MySQL application for managing restaurant menu items with create, update, and delete operations.

## Features
- List menu items
- Add new items
- Update existing items
- Delete menu entries
- Image-based menu presentation
- Vegan / non-vegan item support

## Tech Stack
- PHP
- MySQL
- HTML/CSS
